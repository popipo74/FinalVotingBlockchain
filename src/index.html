How do I use this website for E-voting on the Ethereum Blockchain?

1. Install the MetaMask browser add-on.

2. Ensure that the Ethereum Account has sufficient funds to complete the transaction into MetaMask.

3. Visit the Vote Page and submit your vote.

4. A voter can only vote once.

5. On the Admin pages, Admin can add new candidates.

6. The vote count may be viewed on the result page..
